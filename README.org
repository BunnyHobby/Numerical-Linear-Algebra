- Eigen Setup Instructions
       1. Create an empty *quiz2* /directory/
       2. Go to the Eigen [[http://eigen.tuxfamily.org/index.php?title=Main_Page#Download][download page]]
       3. Download the source code (/.tar.gz/)
       4. Extract the file anywhere
       5. Copy the *Eigen* /subdirectory/ into your *quiz2* /directory/

- Building and Running the Code

       Create a plain text file call *Makefile* in your *quiz2* /directory/.
       
       Type =make= in the *Terminal* and to run it, type =./quiz2=.
